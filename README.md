# projeto_velozes_e_furiosos
FAST SAGA é um projeto web desenvolvido em HTML5, CSS3 e JavaScript puro, inspirado na franquia Velozes e Furiosos. O site apresenta uma interface responsiva e moderna com informações sobre os filmes, curiosidades, galeria e contato, utilizando componentes reutilizáveis (Web Components) e boas práticas de semântica HTML e organização de código.
