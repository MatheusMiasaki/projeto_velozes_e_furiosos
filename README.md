# 🚗 FAST SAGA

FAST SAGA é um projeto acadêmico desenvolvido com HTML5, CSS3 e JavaScript puro, inspirado na franquia **Velozes e Furiosos**.

O objetivo do projeto é aplicar boas práticas de desenvolvimento Front-end, criando um site moderno, responsivo e organizado, utilizando componentes reutilizáveis e HTML semântico.

## ✨ Funcionalidades

- 🎬 Página com todos os filmes da franquia
- 🏎️ Curiosidades e bastidores
- 📸 Galeria de carros icônicos
- 📞 Página de contato
- 📱 Layout responsivo
- ♻️ Componentes reutilizáveis (Header e Footer)
- 🎨 Design System com CSS Variables
- 🌐 HTML5 semântico

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (ES6)
- Web Components
